# 判断浏览器和版本

　　由于不同的浏览器中有些对象接口的使用方式是不同的或者实现一个场景的方式也是不同的，所以需要根据不同的浏览器运行不同的JS脚本。

　　本质上是使用Navigator对象的appName+appVersion属性或者直接使用userAgent属性进行实现。
